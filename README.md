Merge
Merge combines two branches together.
It keeps the complete history of both branches.
Git creates a new merge commit.

Example:

git merge branch1



Rebase
Rebase moves your branch commits on top of another branch.
It creates a cleaner and more linear history.
It does not create a merge commit.

Example:

git rebase main
